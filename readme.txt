== Description ==
Kiniso is a plugin that helps you manage your records and lists.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/kiniso` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the shortcode `[kiniso_list]` to display the list of records on your page.
4. Use the shortcode `[kiniso_form]` to display the form to insert records to the database.

== Frequently Asked Questions ==
